结构体成员变量说明

uint32_t qualityLvl

视频质量最低水平, 范围[0-7], 值越低图像质量越高, 但码流越大.

minBitRate = maxBitRate * quality[qualityLvl], 其中quality[] = {0.8, 0.7, 0.6, 0.5, 0.4, 0.3, 0.2, 0.1}
