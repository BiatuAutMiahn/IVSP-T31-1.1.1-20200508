 ▼IMP(Ingenic Media Platform)
 IMP_System系统控制模块，包括IMP的基础功能以及模块间绑定的相关功能

 IMP_ISP图像信号处理单元。主要包含图像效果设置、模式切换以及Sensor的注册添加删除等操作
 IMP_Encoder视频编码（JPEG, H264, H265）模块，包含编码通道管理，编码参数设置等功能
 IMP_Decoder视频解码模块，当前只支持JPEG解码
 IMP_Audio音频模块，包含录放音、音频编解码、音量及增益设置、回声消除、自动增益等功能
 IMP_OSDOSD模块，可在视频流上叠加图片、位图、直线、矩形框
